# 待办事项应用规格说明

## 1. 项目概述

- **项目名称**: Task Flow
- **项目类型**: 单页 React Web 应用
- **核心功能**: 纯前端的待办事项管理，支持添加、编辑、删除、标记完成，所有数据存储在 localStorage 中
- **目标用户**: 需要简单任务管理的个人用户

## 2. UI/UX 规格

### 布局结构

- **整体布局**: 垂直居中的单列布局，最大宽度 520px
- **页面区域**:
  - 顶部标题区 (app header)
  - 输入区域 (task input)
  - 任务列表区 (task list)
  - 统计信息区 (footer stats)

### 响应式断点

- **移动端**: < 640px - 全宽度 padding: 16px
- **桌面端**: >= 640px - 居中显示，最大宽度 520px

### 视觉设计

#### 色彩方案 (深色主题 - 墨玉绿)

- **背景色**: `#0d1117` (深黑蓝)
- **卡片背景**: `#161b22` (深灰)
- **主色调**: `#58a6ff` (亮蓝)
- **次要色**: `#8b949e` (灰色)
- **成功色**: `#3fb950` (绿色)
- **警告色**: `#d29922` (琥珀色)
- **危险色**: `#f85149` (红色)
- **文字主色**: `#e6edf3` (浅白)
- **文字次色**: `#8b949e` (灰色)
- **边框色**: `#30363d` (深灰)

#### 字体

- **主字体**: `'Crimson Pro', Georgia, serif` (标题和强调文字)
- **正文字体**: `'Source Sans 3', -apple-system, sans-serif`
- **字号**:
  - 应用标题: 2rem (32px), font-weight: 700
  - 任务文本: 1rem (16px)
  - 辅助文字: 0.875rem (14px)

#### 间距系统

- **基础单位**: 8px
- **卡片内边距**: 24px
- **任务项间距**: 12px
- **输入框内边距**: 12px 16px

#### 视觉效果

- **边框圆角**: 12px (卡片), 8px (输入框和按钮)
- **阴影**: `0 8px 24px rgba(0, 0, 0, 0.4)`
- **过渡动画**: 所有交互 0.2s ease
- **完成任务**: 添加删除线，颜色变淡

### 组件规格

#### 1. 应用头部

- 左侧: 应用图标 (✓ 符号) + 标题 "Task Flow"
- 右侧: 主题切换按钮 (可选)

#### 2. 输入区域

- 输入框 placeholder: "添加新任务..."
- 右侧添加按钮 (图标: +)
- 按 Enter 或点击按钮添加
- 输入验证: 空内容不允许添加

#### 3. 任务列表

- **任务项结构**:
  - 左侧: 自定义复选框 (圆形)
  - 中间: 任务文本 (可编辑)
  - 右侧: 编辑按钮 + 删除按钮
  
- **任务状态**:
  - 未完成: 正常显示
  - 完成: 文字划线 + 透明度 0.5
  - 悬停: 显示操作按钮

- **空状态**: 显示 "暂无任务，开始添加吧！" 提示

#### 4. 统计信息区

- 显示: "已完成 X / 共 Y 项"
- 进度条显示完成比例

## 3. 功能规格

### 核心功能

1. **添加任务**
   - 输入框输入内容
   - 可选的截止日期选择器
   - 按 Enter 或点击添加按钮
   - 添加后清空输入框
   - 新任务添加到列表顶部

2. **完成任务**
   - 点击复选框切换完成状态
   - 完成的任务显示删除线

3. **编辑任务**
   - 点击编辑按钮进入编辑模式
   - 输入框变为可编辑状态
   - 可修改截止日期
   - 按 Enter 保存，ESC 取消

4. **删除任务**
   - 点击删除按钮
   - 弹出确认提示 (可选)
   - 从列表中移除

5. **数据持久化**
   - 所有数据存储在 localStorage
   - Key: `taskflow-tasks`
   - 数据格式: JSON 数组
   - 每个任务: `{ id, text, completed, createdAt, dueDate }`
   - dueDate 为可选字段，格式为 ISO 日期字符串 (YYYY-MM-DD)

6. **截止日期显示**
   - 任务项显示截止日期标签
   - 已过期的未完成任务显示警告色
   - 今天的任务显示不同样式

### 用户交互流程

1. 打开应用 → 从 localStorage 加载任务
2. 输入任务 → 点击添加/按 Enter → 任务添加到顶部 → 保存到 localStorage
3. 点击复选框 → 切换完成状态 → 保存到 localStorage
4. 点击编辑 → 进入编辑模式 → 修改后保存
5. 点击删除 → 确认后移除 → 保存到 localStorage

### 边界情况处理

- 空输入: 不允许添加
- 仅有空格: 不允许添加 (trim 后判断)
- localStorage 为空: 初始化空数组
- localStorage 数据损坏: 重置为空数组

## 4. 技术栈

- **框架**: React 18 + Vite
- **样式**: 纯 CSS (不使用 Tailwind)
- **状态管理**: React useState + useEffect
- **存储**: localStorage API
- **构建工具**: Vite
